Navigation

  • AD Keys: fly left/right
  • WS Keys: fly forward/backward
  • RF Keys: fly up/down
  • Click and drag: rotate view

Center Selection

Hold CTRL to enable pivot selection mode. Click on the object you want to center rotation on. Use mouse wheel to scroll forward or backward to move the center of rotation along the view ray. The object will turn greener the closer you are to center on it. When done, release the CTRL button and click anywhere.

User Interface

  • H Key: toggle this help screen
  • C Keys: toggle log
  • M Keys: toggle colormap

Console

The console is a full python interpreter. here are a few useful commands as a reference.
  • :q quit the application
  • resetPivot() set the center of rotation to the center of the model
  • dqDec = [value] set progressive render quality. Try values such as 1,2,8,64
Press H to open the help screen.
